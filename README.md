# Dr. Marwan Hesham Physical Therapy Clinic

A modern, responsive website for Dr. Marwan Hesham's physiotherapy clinic built with Next.js 16, TypeScript, and Tailwind CSS.

## Features

- Fully responsive design
- Multi-page layout (Home, Services, About, Contact)
- Dynamic content from JSON file
- Professional clinic styling
- Interactive components

## Pages

1. **Home Page** - Hero section, services preview, about preview, testimonials
2. **Services Page** - Detailed list of all services
3. **About Page** - Doctor biography and clinic philosophy
4. **Contact Page** - Contact form and clinic information
